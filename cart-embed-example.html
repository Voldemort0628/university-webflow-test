How to Embed in Webflow

  1. Create an HTML embed element in Webflow
  2. Copy the code from the box below
  3. Paste it into the Webflow embed element
  4. Publish your Webflow site

Basic Embed Code

<div id="shopify-cart-root"></div>

<script>
  // Configuration for the cart
  window.shopifyCartConfig = {
    storeDomain: 'hvb9gt-qv.myshopify.com',
    storefrontAccessToken: '897d2510d19bd01af34eda4f2ea9523b',
    products: []
  };
</script>

<script src="https://your-deployment-url/shopify-cart.js"></script>

Advanced Configuration

You can customize the cart behavior with advanced configuration options:

Custom Configuration

<div id="shopify-cart-root"></div>

<script>
  // Advanced configuration for the cart
  window.shopifyCartConfig = {
    // Required Shopify credentials
    storeDomain: 'your-store.myshopify.com',
    storefrontAccessToken: 'your-storefront-access-token',
    
    // Optional: Pre-load products in the cart
    products: [
      {
        id: 'product-handle-1',
        name: 'Product Name',
        price: 99.99,
        image: 'https://your-site.com/image.jpg',
        quantity: 1
      }
    ],
    
    // Optional: Callback when checkout completes
    onCheckoutComplete: function() {
      console.log('Checkout completed!');
      // You can add custom tracking or redirect logic here
    }
  };
</script>

<script src="https://your-deployment-url/shopify-cart.js"></script>

Adding Products Programmatically

You can also add products to cart from anywhere on your page:

<button onclick="addProductToCart()">Add Product</button>

<script>
  function addProductToCart() {
    if (window.GrowlinkShopifyCart) {
      window.GrowlinkShopifyCart.addProduct({
        id: 'product-handle',
        name: 'Product Name',
        price: 99.99,
        image: 'https://your-site.com/image.jpg',
        quantity: 1
      });
    }
  }
</script>
